# Game Modes Catalog

Date: 2026-02-22
Status: Working list (Mayhem removed from scope)

## Implemented / Current

1. Classic Duel (Powerups On)
- Single-player vs AI.
- Core Pong rules with random powerups.
- Manual powerups for player use `E` to trigger.
- Auto powerups can affect either side.

## Candidate Modes (Backlog)

1. Turbo Rally
- Ball speed ramps every N paddle hits.
- Match pressure increases continuously.

2. Mirror Arena
- Left-right controls invert at timed intervals.
- Encourages anticipation and adaptation.

3. Gravity Wells
- Temporary gravity zones bend ball trajectory.
- Spatial control and timing become strategic.

4. Split Ball
- Ball can split into two for short bursts.
- Scoring logic uses first ball to cross goal.

5. Sudden Death
- One point wins after a short setup countdown.
- High-intensity quick rounds.

6. Shield Clash
- Both sides spawn more shield-type manual powerups.
- Longer rallies with defensive timing.

7. Precision Mode
- Smaller paddles, higher control rewards.
- Emphasizes player accuracy and positioning.

## De-Scoped

1. Mayhem Mode
- Removed per latest direction.
- Any future chaos mechanics should be introduced as a separate named mode from this catalog.
