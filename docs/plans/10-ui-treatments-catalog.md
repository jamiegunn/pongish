# UI Treatments Catalog

Date: 2026-02-22
Status: Working list

## Current Default

1. Retro Cabinet (Default)
- Dark arcade shell palette with amber highlights.
- Scanline-inspired arena rendering.
- Cabinet-style HUD and controls.

## Candidate Treatments

1. Tournament Broadcast
- Sports HUD overlays, score ticker, split stat cards.
- Clean high-contrast presentation for spectators.

2. Neon Wireframe
- Black background with bright vector lines.
- Minimal geometry, high motion emphasis.

3. Industrial Console
- Utilitarian paneling, warning accents, numeric displays.
- Mechanical arcade-machine vibe.

4. Synthwave Skyline
- Sunset gradients, horizon grid, chromatic light blooms.
- Stylized and energetic aesthetic.

5. Minimal Lab
- Neutral tones, data-forward UI, reduced ornamentation.
- Focused for training and performance testing.

6. Handheld Pixel
- Chunky pixel styling and compact HUD layout.
- Portable-device inspired look and spacing.

1. Powerup Alert Banner
- Elevated animated banner for each powerup event.

2. Manual Ready Prompt
- Dedicated cue when a manual powerup can be activated.

3. Heatmap Overlay Toggle
- Optional ball-trace intensity layer over the arena.
